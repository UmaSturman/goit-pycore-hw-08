# goit-pycore-hw-08
Address Book CLI
Address Book CLI - це консольний застосунок для зручного управління особистою адресною книгою, написаний на чистому Python без сторонніх залежностей.
Застосунок дозволяє зберігати контакти з іменами, телефонними номерами та датами народження. Всі дані автоматично зберігаються на диск при виході з програми та відновлюються при наступному запуску - завдяки серіалізації через pickle. Це означає, що ви ніколи не втратите свої контакти між сесіями.
Зручний CLI-інтерфейс із вбудованою валідацією введення захищає від помилок: програма перевіряє формат телефону (10 цифр) та дати народження (DD.MM.YYYY), а також автоматично переносить нагадування про день народження з вихідних на понеділок.

Консольний застосунок для управління адресною книгою з підтримкою збереження даних між сесіями.

При запуску програма автоматично завантажує збережені контакти з файлу addressbook.pkl. При виході (exit або close) дані зберігаються автоматично.

Збереження даних
Дані серіалізуються через модуль pickle у файл addressbook.pkl у поточній директорії. Після перезапуску програми адресна книга відновлюється з попереднього сеансу.

Команди
hello - Привітання
add<ім'я> <телефон> - Додати контакт або новий телефон до існуючого
change<ім'я> <старий_тел> <новий_тел> - Змінити номер телефону
phone<ім'я> - Показати контакт
all - Показати всі контакти
add-birthday<ім'я> <дата> - Додати день народження (формат DD.MM.YYYY)
show-birthday<ім'я> - Показати день народження контакту
birthdays - Найближчі дні народження (наступні 7 днів)
exit / close - Зберегти та вийти


 Приклад використання
Welcome to the assistant bot!
Enter a command: add John 1234567890
Contact added.
Enter a command: add-birthday John 15.03.1990
Birthday 15.03.1990 added to John.
Enter a command: phone John
Contact name: John, phones: 1234567890, birthday: 15.03.1990
Enter a command: birthdays
John: 16.03.2026
Enter a command: exit
Good bye!


Структура коду
address_book.py
│
├── Field          # Базовий клас поля
│   ├── Name       # Ім'я контакту (не може бути порожнім)
│   ├── Phone      # Телефон (рівно 10 цифр)
│   └── Birthday   # Дата народження (формат DD.MM.YYYY)
│
├── Record         # Запис контакту (ім'я + телефони + день народження)
│
├── AddressBook    # Колекція контактів (успадковує UserDict)
│
├── save_data()    # Серіалізація книги у pickle-файл
├── load_data()    # Десеріалізація книги з pickle-файлу
│
└── main()         # CLI-цикл програми

 Валідація

Ім'я - не може бути порожнім
Телефон - рівно 10 цифр, тільки цифри
Дата народження - формат DD.MM.YYYY
Якщо день народження випадає на вихідний - дата привітання переноситься на найближчий понеділок

Залежності
Тільки стандартна бібліотека Python 3:

pickle - серіалізація даних
collections.UserDict - базовий клас адресної книги
datetime - робота з датами
